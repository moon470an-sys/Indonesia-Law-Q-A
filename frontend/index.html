기능 안내

  • 법령 코퍼스 카드 클릭 — 검색 범위를 해당 법령으로 한정 (다중 선택). 헤더의 전체 / 해제로 일괄 토글.
  • 예시 질문 칩 — 선택한 카테고리에 맞춰 자동 필터링됩니다.
  • 인용 칩 호버 — 답변 안의 (Pasal X, 출처: 파일명) 칩에 마우스를 올리면 해당 출처 미리보기가 떠오릅니다. 클릭하면 출처 카드로 이동.
  • 과거 질문 클릭 — 입력창에 다시 채워서 변형 질문하기 좋습니다.
  • 📋 복사 / 📥 내보내기 — 단일 답변 클립보드 복사, 필터된 히스토리 전체 Markdown 저장.
  • 출처 카테고리 필터 — Q&A 카드 안의 작은 칩으로 출처를 카테고리별로 토글.
  • 히스토리 검색 — 저장된 모든 Q&A에서 텍스트 검색 (매칭 부분 자동 하이라이트).
  • 인쇄/PDF — Ctrl+P 로 깔끔한 보고서 형태로 출력 (UI 청크 자동 숨김).
